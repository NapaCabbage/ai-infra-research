# AI Infra Research Vault

个人 AI 基础设施研究笔记库，基于 Obsidian 构建。

## 内容概览

系统性学习 AI Infra 核心方向，覆盖推理优化、训练系统、GPU 架构等领域的前沿论文与工程实践。

**主要研究方向：**
- **Inference Serving**：Orca、vLLM、SGLang、DistServe、Mooncake
- **Attention 优化**：FlashAttention、SageAttention
- **Training 系统**：Megatron-LM、3D Parallelism、Activation Recomputation、DeepSeek-V3
- **RL Training**：PPO、GRPO、DeepSeek-R1
- **GPU 架构**：SM/Warp/CUDA Core、Memory Hierarchy、Tensor Core
